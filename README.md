# Driving Simulator

3D driving simulator built with Godot 4.6 (.NET).

## Features
- 3 selectable cars: Lamborghini Huracán, Ferrari SF90, Bugatti Chiron
- Open city map with roads, buildings and procedural environment
- Task system with score (drive to gas station, city hall, police, hospital, return)
- HUD with speedometer, current task and score
- Realistic vehicle physics using Godot's VehicleBody3D + Jolt Physics

## Controls
- **W / S** — Accelerate / Reverse
- **A / D** — Steer
- **SPACE** — Handbrake
- **ESC** — Back to menu

## Run
Open the project in Godot 4.6+ and press F5.
